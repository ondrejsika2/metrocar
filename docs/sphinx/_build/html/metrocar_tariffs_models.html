metrocar.tariffs.models¶

class metrocar.tariffs.models.FixedPaymentTariff(*args, **kwargs)¶

Represents user tariff which allows users to drive for less if they use carsharing services often.

It consists of several benefits:
  • free kms for month
  • correction ratio for regular price which lowers the regular prices
